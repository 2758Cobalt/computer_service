# coursework databases and knowledge
A coursework application that provides the ability to work with the computer service database.

Kolomoyets S.V. Development of an application for automation of business processes of a computer service with the use of modern technologies and visual design tools with elements of a user-friendly interface for operation under the Windows operating system. - Course work on the rights of the manuscript.
Course work in the discipline "Databases and Knowledge", field of knowledge 12 "Information Technology", speciality 121 "Software Engineering" - University of Customs and Finance, Dnipro, 2023.
As a result of the course work, a structure was developed, constructors and methods for setting and displaying data field values were defined, methods for searching and analysing data were implemented to control the work of a computer service in the Python programming language using interface visualisation libraries in the Microsoft Visual Studio Code development environment.
